# Cyber Task 11: Phishing Attack Simulation & Detection

## Objective
To understand phishing attacks by simulating a fake phishing email and analyzing how to detect and prevent them.

## What is Phishing?
Phishing is a social engineering attack where attackers trick users into revealing sensitive information.

## Simulation Performed
- Created a fake phishing email template
- Designed a fake login landing page
- Identified phishing red flags
- Studied prevention techniques

## Red Flags Identified
- Urgent message
- Suspicious link
- Generic greeting
- Credential request

## Prevention Methods
- Verify sender identity
- Avoid clicking unknown links
- Use multi-factor authentication
- Security awareness training

## Tools Used
- Text editor
- Web browser
- Manual phishing simulation

## Outcome
Improved understanding of phishing attacks and how to detect and prevent them.
